## ğŸ“Š Database Query Visualization

### Main Timeline Query

```sql
-- Simplified version of the query
SELECT 
  order_no,                              -- Group identifier
  order_origin,
  trx_type,
  ship_from, ship_from_desc,
  ship_to, ship_to_desc,
  MIN(order_date) as order_date,         -- Start point
  MIN(delivery_date) as planned_date,    -- Planned milestone
  MAX(receipt_date) as actual_date,      -- End point
  COUNT(*) as total_lines,
  SUM(order_qty) as total_order_qty,
  SUM(ship_qty) as total_ship_qty,
  CASE 
    WHEN MAX(receipt_date) IS NULL 
      THEN 'pending'
    WHEN MAX(receipt_date) <= MIN(delivery_date) 
      THEN 'on_time'
    ELSE 'delayed'
  END as delivery_status
FROM view_warehouse_order_line
WHERE order_date >= '2025-10-01'
  AND order_date <= '2025-10-31'
GROUP BY order_no, order_origin, trx_type, 
         ship_from, ship_from_desc, ship_to, ship_to_desc
ORDER BY order_date DESC
LIMIT 50
```
