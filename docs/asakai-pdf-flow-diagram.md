### Example 2: Filter by Section + Monthly Period
```
REQUEST: ?section=brazzing&period=monthly&date_from=2024-01-01

SQL WHERE: 
  section = 'brazzing' 
  AND YEAR(date) = 2024

RESULT: 
  - Semua data section brazzing di tahun 2024
  - PDF Header DEPT: BRAZZING
  - PDF Header BULAN: January, TAHUN: 2024
```

```
REQUEST: ?period=daily&date_from=2024-01-15

  YEAR(date) = 2024

RESULT: Semua data di bulan Januari 2024
```
